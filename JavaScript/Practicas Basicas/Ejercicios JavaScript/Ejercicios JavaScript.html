Ejercicios en JavaScript

Ejercicio 1 - Escribe un programa que pida una frase y escriba cuántas veces aparecen cada una de las vocales



Ejercicio 2 - Escribe un programa que solicite al usuario ingresar el número de kilómetros recorridos por su coche y el número de litros consumidos.
El script debe mostrar el consumo de combustible por kilómetro.





Ejercicio 3 - Escribe un programa que solicite al usuario ingresar las horas y los minutos, mostrar los segundos de ese tiempo.





Ejercicio 4 - Escribe un programa que solicite al usuario ingresar un número de 2 dígitos y muestre el número de unidades y de decenas de dicho número



Ejercicio 5 - Una tortilla de patatas lleva 200 gramos de patatas por persona.
Por cada kilo de patatas se necesitan 5 huevos y 300 gramos de cebolla.
Escribe un script que dado el número de comensales calcule las cantidades de ingredientes necesarias



Ejercicio 6 - Crear en javascript un programa que tome un array de números y me muestre el número más grande del array



Ejercicio 7 - Crear en javascript un programa que tome un array de números y me muestre la suma total de todos los números del array



Ejercicio 8 - El factorial de un número entero n es una operación matemática que consiste en multiplicar todos los factores n x (n-1) x (n-2) x ... x 1.
Así, el factorial de 5 (escrito como 5!) es igual a: 5! = 5 x 4 x 3 x 2 x 1 = 120
Utilizando la estructura for, crear un script que calcule el factorial de un número entero.



Ejercicio 10 - Definir una función que determine si la cadena de texto que se le pasa como parámetro es un palíndromo
Es decir, si se lee de la misma forma desde la izquierda y desde la derecha.
Ejemplo de palíndromo simple:"oso"
Ejemplo de palíndromo complejo: "La ruta nos aportó otro paso natural".



Ejercicio 11 - Escribir una función llamada removerCeros que reciba un arreglo de números y retorne un nuevo arreglo excluyendo los ceros (0).
